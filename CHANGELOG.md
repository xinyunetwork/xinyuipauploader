# 更新记录

## 0.2.3 - 2026-07-07

- 注销后可检测并选择本机仍有效的付费授权，无需再次寻找授权码
- 客户端增加购买授权与套餐价格查看入口
- 首页欢迎标题和介绍文字支持在授权后台编辑
- 证书操作合并为下拉菜单，修改 P12 密码移动到每张证书的操作菜单

## 0.2.2 - 2026-07-06

- 首次启动直接进入软件，不再要求填写凭证或弹出激活窗口
- 每次手动添加 API 账号后，后台静默校验该账号自己的 3 天试用
- 已用过试用的账号只拦截自身，其他账号的试用和使用不受影响
- 付费授权绑定整机，授权后可使用全部账号

## 0.2.1 - 2026-07-06

- 激活与试用不再要求手填 Apple ID/Issuer ID
- 自动记录全部已保存 Apple ID、Issuer ID 与 Key ID 的不可逆指纹
- 阻止换电脑导入任一旧凭证重复领取试用
- 首页显示套餐名称、到期时间和剩余天数
- API Key 模式不再无效查询 Provider Short Name
- Apple ID 模式 Provider 查询增加 45 秒超时与错误回收

## 0.2.0 - 2026-07-06

- 接入在线商业授权与三天免费试用
- 增加日卡、月卡、年卡和单机永久授权
- Apple 凭证改用 Windows DPAPI 加密保存
- 修复 Provider Short Name 获取与解析
- 自动检测 Apple Transporter，缺失时引导官方下载
- 增加 JavaScript 混淆、ASAR 完整性与 Electron Fuses
- 增加应用自动更新

## 0.1.0

- 首个内部测试版本
